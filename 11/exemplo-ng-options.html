Diretiva NG-OPTION

Maneiras de Usar o ng-option

LABEL FOR IN ARRAY

Este retorna o objeto inteiro

Resultado: {{VmAngularOption.lista_dados}}


SELECT AS VALUE LABEL IN ARRAY

Este é o mais usado no mercado

Resultado: {{VmAngularOption.lista_dados_com_valor | lowercase}}


SELECT AS VALUE LABEL IN ARRAY COM VALOR INICIAL

Aqui no já iniciamos o option com um valor inicial setado

Resultado: {{VmAngularOption.lista_dados_com_valor_inicial | lowercase}}


SELECT AS VALUE LABEL IN ARRAY USANDO GROUPY BY

Aqui nos podemos selecionar um cliente e agrupa-lo por um tipo especifico

Resultado: {{VmAngularOption.lista_dados_groupby.Email | lowercase}}


SELECT AS VALUE LABEL IN ARRAY USANDO GROUPY BY E DISABLE

Aqui nos podemos selecionar um cliente e agrupa-lo por um tipo especifico e desabilitando os itens excluidos

Resultado: {{VmAngularOption.lista_dados_groupby_disable.Email | lowercase}}
